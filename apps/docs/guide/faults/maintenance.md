# 维保说明书管理

![维保说明书管理](/screenshots/admin-maintenance.png)

## 界面布局

页面分为左右两个面板：

### 左面板：说明书文件列表

- 列出已上传的维保说明书文件
- 每个文件项显示文件名和上传日期
- 支持操作：
  - **点击**文件：在新标签页中预览 PDF
  - **双击文件名**：进入内联重命名模式
    - Enter 确认重命名
    - Escape 取消
  - **删除按钮**：弹出确认对话框后删除文件
  - **上传按钮**：选择本地 PDF 文件上传

### 上传多份 PDF

- 支持一次上传多份 PDF 文件
- 限制：单文件不超过 100MB
- 上传进度条显示
- 上传完成后文件列表自动刷新

### 右面板：日历月视图

- 显示当前月份的日历
- 日期上标记当天关联的维护事件（如有）
- 点击日期切换查看当天的维护记录
- 使用 `react-day-picker` 实现

## 数据存储

- 文件存储在后端配置的存储后端中（开发环境为本地磁盘，生产环境为 MinIO S3 对象存储）
- 元数据存储在 `files` 表（文件名、路径、大小、上传时间）
- 维护记录存储在 `maintenance` 表
