# 实时故障监控

![实时故障监控](/screenshots/admin-faults.png)

## 故障列表

页面以表格形式展示所有故障/告警记录：

| 字段     | 说明                        |
| -------- | --------------------------- |
| 设备名   | 触发故障的设备              |
| 故障等级 | 🔴 紧急 / 🟡 警告 / 🔵 信息 |
| 故障时间 | 触发时间戳                  |
| 状态     | 待处理 / 处理中 / 已解决    |

### 筛选

页面顶部提供筛选栏：

- **时间范围**：最近 1 小时 / 今天 / 本周 / 本月 / 自定义
- **故障等级**：多选筛选项
- **设备名称**：文本搜索

### 快速跳转

从 [管理总览](/guide/dashboard/admin-overview) 顶部的「未处理告警」区域点击「查看全部」可直接跳转到此页，并自动应用当前筛选。

## 操作

列表中每条记录支持：

- **点击行**：展开故障详情（设备信息、故障描述、建议处理方案）
- **状态变更**：处理中 → 已解决（确认处理后更新状态）

## 数据源

故障数据来自 `faults` 表和 `alerts` 表的联合查询：

- `faults`：设备故障记录
- `alerts`：系统告警规则触发的告警
