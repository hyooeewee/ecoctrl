## 统计卡片

页面顶部显示统计摘要：

| 指标                  | 说明                               |
| --------------------- | ---------------------------------- |
| 📊 本月总故障数       | 当前月份累计故障数量               |
| ⏱ 平均修复时间 (MTTR) | 从「处理中」到「已解决」的平均耗时 |
| 🔄 同比变化           | 与上月相比的变化百分比             |

## 故障趋势图

页面中部展示故障数量趋势：

- X 轴：日期（最近 30 天）
- Y 轴：故障数量
- 折线图显示每日新增故障数
- 可切换时间维度（按日 / 按周）

## 按等级分布

饼图展示本月故障的等级分布：

- 🔴 紧急故障占比
- 🟡 告警占比
- 🔵 信息类占比

各等级带颜色编码，便于直观识别。

## 数据源

统计数据来自 `fault_stats` 表的聚合查询：

- 故障数量通过 `faults` 表实时统计
- MTTR 通过各故障记录的处理时间差值计算
- 趋势数据按时间范围聚合
