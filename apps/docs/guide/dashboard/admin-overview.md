# Admin 管理总览

![Admin 管理总览](/screenshots/admin-overview.png)

登录 Admin 后台后，默认进入**管理总览**页面。这里是平台的指挥中心，集中展示核心运行指标。

## 指标卡片

页面顶部是四张指标卡片，每张显示当前数值与环比变化趋势：

| 卡片          | 指标                    | 数据来源               |
| ------------- | ----------------------- | ---------------------- |
| ⚡ 总能耗     | 当前周期总耗电量（kWh） | `energy_readings` 聚合 |
| 📡 设备在线率 | 在线设备数 / 总设备数   | IoT 心跳检测           |
| 🔴 未处理告警 | 当前未处理的故障/告警数 | `alerts` + `faults` 表 |
| 🌿 本月碳排   | 本月碳排放总量（tCO₂）  | 能耗 × 碳排放因子计算  |

每张卡片右侧显示**趋势箭头**（↑ 上升 / ↓ 下降）和百分比变化，反映与上一周期的对比。

## 能耗趋势图

页面中部是**近一周能耗趋势**面积图：

- X 轴：日期（最近 7 天）
- Y 轴：能耗值（kWh）
- 填充区域表示逐日变化趋势
- 支持切换显示粒度（日 / 时）

## 实时告警列表

右侧面板列出最近的未处理告警：

| 字段   | 说明                        |
| ------ | --------------------------- |
| 设备名 | 触发告警的设备名称          |
| 级别   | 🔴 紧急 / 🟡 警告 / 🔵 信息 |
| 时间   | 告警触发时间                |
| 状态   | 待处理 / 处理中 / 已解决    |

点击「查看全部」跳转到 [实时故障监控](/guide/faults/real-time) 页面。

## 系统状态

页面底部系统状态区域显示：

- **API 服务状态**：在线 / 离线
- **数据库连接**：正常 / 异常
- **IoT 网关**：已连接 / 断开（含图标指示）
- **下次自动备份时间**：根据 `backupSchedule` 配置显示

> 提示：页面数据通过 REST API 定时刷新。刷新间隔可在 [系统配置](/guide/account/system-config) 中调整（10s / 30s / 1m / 5m）。
