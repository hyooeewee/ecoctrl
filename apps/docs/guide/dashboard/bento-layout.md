# 编辑仪表盘布局

![编辑仪表盘布局](/screenshots/bento-layout.png)

Web 门户的仪表盘使用 **Bento Grid** 布局引擎，支持自由调整小组件的位置和尺寸。

## 进入编辑模式

通过以下方式进入编辑模式：

1. 导航到 **设置 → 布局**，点击「编辑仪表盘布局」
2. 编辑模式下组件的边框变为虚线，左上角出现操作手柄

## 拖拽重排

进入编辑模式后：

- **拖拽标题区域**：移动组件到网格中的其他位置
- 其他组件会自动让位并重新排列
- 释放后布局立即生效

## 调整尺寸

选中组件后，右下角出现尺寸调节手柄：

- 拖拽手柄调整组件的宽高
- 网格基于 CSS Grid 实现，尺寸单位是网格单元格
- 组件宽度范围：1-4 网格单元

## 隐藏/显示组件

编辑模式下每个组件右上角有隐藏按钮：

- 点击后组件折叠为一个占位条
- 再次点击恢复显示
- 隐藏的组件在刷新后保持隐藏状态

## 自动退出超时

编辑模式空闲一段时间后自动退出（防止误操作）：

| 可选超时 | 适用场景                 |
| -------- | ------------------------ |
| 15 秒    | 快速调整                 |
| 30 秒    | 常规编辑                 |
| 1 分钟   | 仔细排版                 |
| 5 分钟   | 复杂布局                 |
| 永不过期 | 长时间编辑（需手动退出） |

可在 **设置 → 布局 → 自动退出编辑延迟** 中配置。

## 恢复默认布局

点击「恢复默认布局」按钮：

1. 弹出确认对话框
2. 确认后所有组件回到初始位置和尺寸
3. 隐藏的组件重新显示
4. 操作不可撤销

## 布局持久化

当前布局自动保存到服务器（`dashboard_widgets` 表），与用户账户绑定。切换设备后同步加载，无需重新配置。
