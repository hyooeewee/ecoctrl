# 业务对象管理

![业务对象管理](/screenshots/admin-objects.png)

数据模型页面的第二个子 Tab，管理业务对象（Business Object）。

## 对象列表

以表格形式展示业务对象：

| 字段     | 说明                             |
| -------- | -------------------------------- |
| 编码     | 业务对象编码（全局唯一）         |
| 名称     | 对象名称                         |
| 关联模型 | 所属数据模型，显示模型名称和版本 |

### 搜索筛选

搜索框支持按编码或名称模糊匹配。

## 新增业务对象

点击「新增对象」按钮打开对话框：

| 字段     | 必填 | 说明                   |
| -------- | ---- | ---------------------- |
| 选择模型 | 是   | 从已有数据模型下拉选择 |
| 编码     | 是   | 唯一编码，不可重复     |
| 名称     | 是   | 对象名称               |
| 描述     | 否   | 对象说明               |

## 编辑与删除

- **编辑**：点击行右侧编辑按钮，弹窗修改
- **删除**：点击删除按钮 → 确认对话框「此操作不可撤销，关联的业务对象将被清除」

## 业务对象与模型的关系

一个数据模型可以关联多个业务对象。例如：

- 模型：「AHU-01 空调机组」
- 业务对象：「A区 AHU-01」「B区 AHU-02」等

来自 `objects` 表，通过 `modelId` 关联到 `models` 表。
