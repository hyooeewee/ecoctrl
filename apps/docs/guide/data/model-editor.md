# 3D 模型编辑器

![3D 模型编辑器](/screenshots/admin-dashboard-model.png)

「大屏模型」Tab（DashboardModel）是一个全屏沉浸式页面入口，点击后进入全屏场景编辑器。

## 编辑模式

页面进入全屏模式（隐藏侧边栏和头部），右上角提供「返回」按钮回到数据模型页。

编辑器有三种模式，通过模式切换器选择：

### 🏗 模型模式

- 管理已上传 3D 模型文件的排列
- 使用拖拽排序功能调整模型在场景中的加载顺序
- 支持上传新的模型文件

### 🏷 标签模式

- 在 3D 场景中配置热点标签
- 每个标签绑定到一个位置（x, y, z 坐标）
- 标签名称、颜色、关联的区域/数据可编辑

### 🎬 动作模式

- 配置 3D 场景中的交互动作
- 点击热点标签时的行为（飞至、弹窗、数据面板）
- 相机运动的路径和速度

## 上传模型

| 设置项           | 值                                  |
| ---------------- | ----------------------------------- |
| 支持格式         | .glb / .gltf / .obj                 |
| 每模型文件数上限 | 最多 10 个文件                      |
| 单文件限制       | 100MB                               |
| 上传区域         | 支持拖拽文件到上传区 + 点击选择文件 |

上传完成后模型出现在文件列表中，可拖拽排序。

## 文件管理

- **拖拽排序**：调整模型文件在场景中的加载顺序
- **删除文件**：移除不需要的模型
- **自动保存**：修改后配置自动保存到 `dashboard_models` 表
