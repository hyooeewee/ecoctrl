# 数据模型管理

![数据模型管理](/screenshots/admin-models.png)

数据模型页面的第一个子 Tab，管理 3D 模型文件。

## 模型卡片网格

页面以卡片网格形式展示所有上传的 3D 模型：

- 每张卡片显示模型名称和缩略图（如果没有缩略图则显示模型图标）
- 卡片右上角显示版本号标签

### 操作

| 操作     | 说明                                                                    |
| -------- | ----------------------------------------------------------------------- |
| 点击卡片 | 跳转到关联的 [业务对象](/guide/data/objects) 页面（该模型下的对象列表） |
| 预览按钮 | 打开 **ModelViewer** 弹窗，在浏览器中渲染 3D 模型预览                   |
| 编辑按钮 | 编辑模型信息（名称、版本、编码、描述）                                  |
| 删除按钮 | 弹出确认对话框后删除模型                                                |

### ModelViewer 弹窗

- 全屏弹窗使用 Babylon.js 渲染
- 支持旋转、缩放查看模型
- 自动适配模型尺寸到视口

## 模型数据

每个模型包含：名称、版本、编码、文件路径、描述

数据来源：`models` 表。
