# 用户账户管理

![用户账户管理](/screenshots/admin-accounts.png)

账户控制页面（Accounts）用于管理平台所有用户。

## 用户列表

以表格形式展示用户数据：

| 字段   | 说明                                                      |
| ------ | --------------------------------------------------------- |
| 用户名 | 登录使用的用户名                                          |
| 邮箱   | 用户绑定的邮箱地址                                        |
| 角色   | 角色标签（admin / manager / operator / viewer），颜色区分 |
| 状态   | 🟢 在线 / ⚪ 离线（基于活跃会话检测）                     |

### 搜索筛选

页面顶部搜索框支持按以下维度筛选：

- **用户名**：模糊匹配
- **邮箱**：模糊匹配
- **角色**：下拉选择角色类型

## 新增用户

点击「新增用户」按钮打开弹窗：

| 字段   | 必填 | 说明                     |
| ------ | ---- | ------------------------ |
| 用户名 | 是   | 登录用户名，全局唯一     |
| 邮箱   | 是   | 用户邮箱，用于验证码发送 |
| 密码   | 是   | 登录密码                 |
| 角色   | 是   | 从角色列表中选择         |

## 编辑用户

点击用户行打开 **UserSheet** 滑出面板（右侧滑入 400px）：

- **用户名**：可编辑
- **邮箱**：可编辑
- **角色**：下拉选择（admin / manager / operator / viewer）
- **状态**：启用/禁用切换（禁用后用户无法登录）
- 修改后自动保存

## 删除用户

点击删除按钮：

1. 弹出确认对话框「确定要删除用户 [用户名] 吗？此操作不可撤销。」
2. 确认后执行删除
3. 关联的 Refresh Tokens 和 OAuth 绑定同时清除

## 角色说明

| 角色       | 权限范围                               |
| ---------- | -------------------------------------- |
| `admin`    | 全部功能，包括用户管理、系统配置       |
| `manager`  | 运维管理功能，不包含用户管理和系统配置 |
| `operator` | 日常操作功能，故障处理、报表查看       |
| `viewer`   | 只读查看仪表盘和基础数据               |
