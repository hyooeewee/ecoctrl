# 偏好设置

![偏好设置](/screenshots/web-settings.png)

## 外观

| 设置项   | 选项                   | 说明                                                     |
| -------- | ---------------------- | -------------------------------------------------------- |
| 主题     | 浅色 / 深色 / 跟随系统 | 基于 `useColorMode` 实现，选择跟随系统时自动匹配 OS 设置 |
| 界面密度 | 紧凑 / 舒适 / 宽松     | 控制表格行高、间距和元素密度                             |
| 字体大小 | 小 / 中 / 大           | 整体界面字号缩放                                         |

## 语言

- **简体中文**：默认语言
- **English**：开发中（Web 门户已完成，Admin 后台部分文案仍为中文）

## 通知

| 设置项     | 说明                     |
| ---------- | ------------------------ |
| 桌面通知   | 开启后浏览器推送系统通知 |
| 告警提示音 | 新告警时播放提示音       |
| 邮件通知   | 通过 SMTP 发送通知邮件   |

## 界面行为

| 设置项         | 说明                           |
| -------------- | ------------------------------ |
| 默认折叠侧边栏 | 登录后侧边栏默认折叠为图标模式 |
| 面包屑导航     | 开启后页面顶部显示面包屑路径   |

## 保存机制

- 所有偏好设置更改 **500ms 防抖自动保存**
- 无需手动点击保存按钮
- 设置存储在两个位置：
  - `localStorage`：立即生效
  - `userSettings` API 端点：持久化到服务器

## 恢复默认

点击「恢复默认」按钮：

1. 弹出确认对话框
2. 确认后所有偏好设置重置为系统默认值
3. 页面自动刷新
