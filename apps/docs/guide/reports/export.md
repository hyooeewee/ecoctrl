# 数据导出

![数据导出](/screenshots/admin-reports.png)

## 导出当前数据

在能耗管理的 [详细数据](/guide/energy/detailed-data) 和 [统计报表](/guide/energy/stats) 页面中，点击「导出当前数据」按钮。

### 导出格式

- **格式**：CSV（逗号分隔值）
- **编码**：UTF-8 with BOM（兼容 Excel 中文打开）
- **文件名**：自动包含数据范围（如 `energy-data_2026-07-01_2026-07-07.csv`）

### 导出内容

导出数据包含当前筛选条件下所有可见数据列：

| 列       | 说明           |
| -------- | -------------- |
| 时间     | 数据记录时间戳 |
| 区域     | 所属区域       |
| 能耗值   | 耗电量（kWh）  |
| 功率因素 | 实时功率因素   |
| 负载率   | 负载百分比     |

## 数据完整性

- 导出数据不包含分页限制（当前页 + 所有页）
- 时间范围限制仍然生效
- 导出前可预览数据范围
