# 定时报表计划

![定时报表计划](/screenshots/admin-reports.png)

## 计划列表

报表管理页面以列表形式展示所有定时报表计划：

| 字段     | 说明                 |
| -------- | -------------------- |
| 报表名称 | 计划名称，自定义     |
| 收件人   | 报表发送的目标邮箱   |
| 发送频率 | 每天 / 每周 / 每月   |
| 状态开关 | Switch 切换启用/禁用 |

## 新建报表计划

点击「新建计划」按钮打开对话框：

| 字段       | 必填 | 说明                                        |
| ---------- | ---- | ------------------------------------------- |
| 报表名称   | 是   | 计划名称                                    |
| 收件人邮箱 | 是   | CSV 格式支持多个邮箱                        |
| 发送频率   | 是   | 每天 / 每周（指定星期几）/ 每月（指定日期） |
| 状态       | 否   | 新建后默认启用                              |

## 模板管理

点击「管理报表模板」打开 **TemplateDialog**：

- 模板列表：展示已创建的模板（名称、格式、最后修改时间）
- 新建模板：定义报表的内容结构、数据源和布局
- 模板格式：支持表格和图表两种布局
- 模板与计划关联：一个模板可被多个计划使用

## 技术说明

- 报表生成通过 pg-boss 队列异步执行
- 定时发送依赖队列调度器
- 生成完成后自动通过 SMTP 发送到目标邮箱
