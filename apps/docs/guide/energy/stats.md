# 能耗统计报表

![能耗统计报表](/screenshots/admin-energy-stats.png)

## 报表功能

能耗管理第三个子 Tab 提供自定义统计报表。

### 报表参数

| 参数     | 选项                                         |
| -------- | -------------------------------------------- |
| 时间范围 | 自定义起止日期                               |
| 聚合粒度 | 按日 / 按周 / 按月                           |
| 区域     | 全部区域或指定区域                           |
| 指标类型 | 能耗(kWh) / 碳排放(tCO₂) / 功率因素 / 负载率 |

### 报表展示

生成的报表以图表和表格两种方式展示：

- **趋势折线图**：显示指标在时间范围内的变化趋势
- **数据表格**：详细数值（日期、值、环比变化率）
- **汇总行**：合计值、平均值、最大值、最小值

### 报表解读

- **环比分析**：表格中自动计算每行数据的环比变化百分比
- **异常标记**：超过阈值的数据点高亮显示
- **空值处理**：无数据的日期自动跳过，不中断趋势线

### 导出

报表数据支持导出为 CSV 格式（与详细数据导出机制相同）。
