# 碳排放因子管理

![碳排放因子管理](/screenshots/admin-energy-carbon.png)

能耗管理的第四个（最后一个）子 Tab 提供碳排放因子的管理功能。

## 界面布局

页面分为左右两个面板：

### 左面板：因子分类树

- 可展开/折叠的分类树结构，按能源类型组织
- 支持搜索筛选（按名称或分类）
- 分类节点包括：电力、天然气、水、蒸汽等
- 点击分类节点，右面板显示该分类下的因子详情

### 右面板：因子详情表格

每行因子包含：

| 字段     | 说明                       |
| -------- | -------------------------- |
| 因子名称 | 如「华东区域电网排放因子」 |
| 分类     | 所属分类                   |
| 因子值   | 排放系数（如 kgCO₂/kWh）   |
| 单位     | 如 kgCO₂/kWh、tCO₂/万Nm³   |
| 来源     | 数据来源说明               |

## 数据刷新

页面提供「从国家温室气体排放因子库刷新」按钮：

1. 点击后系统请求最新的排放因子数据
2. 增量同步：新增的因子自动插入，已有的因子覆盖更新
3. 同步完成后页面自动刷新

### 与能耗模块的联动

碳排放因子是能耗数据计算碳排放的基础：

```
碳排放量 = 能耗值 × 碳排放因子（按区域/按能源类型）
```

更新因子后，历史碳排放数据会自动重新计算，管理总览页面的碳排卡片会反映最新值。

## 技术说明

- 因子数据存储在 `carbonFactorNodes`（分类树）和 `carbonFactors`（因子值）两张表中
- 分类树支持多级嵌套
- 系统通过 API 与外部排放因子数据库通信（需网络连通）
