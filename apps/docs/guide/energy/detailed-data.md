# 详细能耗数据

![详细能耗数据](/screenshots/admin-energy-details.png)

## 数据视图

能耗管理的第二个子 Tab 提供细粒度的能耗数据表格视图。

### 数据列

| 列           | 说明                 |
| ------------ | -------------------- |
| 时间         | 数据记录的时间戳     |
| 区域         | 所属区域             |
| 能耗值 (kWh) | 该时间点的读数值     |
| 功率因素     | 该时间点的 cosφ      |
| 负载率       | 该时间点的负载百分比 |

### 筛选条件

页面顶部提供筛选栏：

- **时间范围**：支持预设（今天 / 本周 / 本月 / 自定义）
- **区域**：按区域名称筛选
- **粒度**：按时间聚合（5 分钟 / 15 分钟 / 1 小时 / 1 天）

### 数据导出

点击「导出当前数据」按钮，将当前筛选条件下的数据导出为 CSV 格式：

- 文件名自动包含时间范围（如 `energy-data_2026-07-01_2026-07-07.csv`）
- 编码为 UTF-8 with BOM，兼容 Excel 中文打开
- 导出内容包含筛选条件下的完整数据列

### 排序与分页

- 默认按时间降序排列（最新数据在前）
- 支持点击列头切换排序方向
- 后端分页，每页 50 条记录
