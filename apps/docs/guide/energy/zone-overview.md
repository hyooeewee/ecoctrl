# 分区域能耗总览

![分区域能耗总览](/screenshots/admin-energy-zone.png)

能耗管理页面的第一个子 Tab 展示建筑各区域的能耗概览。

## 区域能耗卡片

页面上以卡片网格形式展示各区域能耗数据，每张卡片包含：

| 字段        | 说明                                                  |
| ----------- | ----------------------------------------------------- |
| 🏢 区域名称 | 区域标题，如「A 区」「B 区」「C 区」                  |
| ⚡ 当前值   | 当前周期的总能耗（kWh）                               |
| 🎯 目标值   | 该区域的能耗目标（kWh）                               |
| 📊 进度条   | 当前值 / 目标值的百分比进度，颜色渐变（绿 → 黄 → 红） |
| 📐 功率因素 | cosφ 实时值，反映电能利用效率                         |
| ⚙️ 负载率   | 当前负载 / 额定负载百分比                             |

### 超目标告警

当区域能耗超过目标值时，卡片右上角出现告警 Badge：

- 🔴 红色：超过目标值 110% 以上
- 🟡 黄色：超过目标值但未达 110%
- 🟢 绿色：在目标范围内

点击卡片进入该区域的详细数据页。

## 数据源

区域数据来自 `energy_areas` 表的配置（目标值、功率因素基准等）和 `energy_readings` 表的实时聚合。

- 区域配置在 Admin 后台通过数据库初始化（当前无单独管理界面）
- 目标值通过 seed 数据或 API 更新
