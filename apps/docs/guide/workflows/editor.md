# 可视化工作流编辑器

![可视化工作流编辑器](/screenshots/admin-workflow-editor.png)

点击工作流行或新建工作流后，进入全屏沉浸式编辑器，基于 **ReactFlow（XYFlow）** 实现。

## 界面布局

- **全屏模式**：隐藏侧边栏和头部，最大画布空间
- 左上角：返回按钮 + 工作流名称（可内联编辑）
- 右上角：保存/另存为按钮
- 左侧面板：**节点库** — 可拖拽的节点类型列表
- 中央：空白画布

## 创建流程

1. 从左侧节点库拖拽节点到画布
2. 从一个节点的输出手柄拖拽到另一个节点的输入手柄，创建连线
3. 唯一约束：必须有且仅有一个 Start 节点和一个 End 节点
4. 连线不允许形成闭环（DAG 约束）

## 编辑工作流名称

- 点击工作流名称进入内联编辑模式
- Enter 确认修改
- Escape 取消修改

## 保存

- 点击保存按钮将当前工作流 DSL 持久化到 `workflows` 表
- 保存后版本号递增
- 未保存的更改有离开提示
